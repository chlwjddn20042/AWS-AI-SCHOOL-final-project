# Mindful Pocket (Vue 3 + Vite)

모바일 웹앱용 성향 분석/퀘스트/챗봇 SPA입니다.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173`로 접속합니다.

## 간단 구조

- `src/router`: 라우팅 + 인증 가드
- `src/stores`: Pinia 상태 관리
- `src/services`: Mock 서비스 레이어
- `src/components`: 공통 UI 컴포넌트
- `src/layouts`: 레이아웃
- `src/views`: 화면 단위
